📌 Project Overview

Amazon Reviews Sentiment Analysis is a Natural Language Processing (NLP) project that analyzes customer reviews to determine whether the sentiment is Positive, Negative, or Neutral.

The project uses text preprocessing techniques and machine learning models to classify review sentiments and generate insights that help businesses understand customer feedback.

🎯 Project Objectives

Analyze customer reviews from Amazon dataset

Perform text cleaning and preprocessing

Convert text data into numerical features

Build sentiment classification model

Evaluate model performance

Extract actionable insights from customer feedback
